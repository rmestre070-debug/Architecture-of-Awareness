# Contributor Code of Conduct

## Our Pledge
We are committed to fostering an open, welcoming, and collaborative environment.  
All contributors, maintainers, and users agree to treat each other with respect and professionalism.

---

## Our Standards

### Positive Behaviors
- Being respectful and considerate  
- Offering constructive feedback  
- Accepting responsibility for mistakes  
- Focusing on what is best for the community  
- Helping others grow and succeed  

### Unacceptable Behaviors
- Harassment or discrimination of any kind  
- Personal attacks or insults  
- Trolling or inflammatory comments  
- Publishing private information without permission  
- Any behavior that disrupts project collaboration  

---

## Our Responsibilities
Project maintainers are responsible for:

- Clarifying standards of acceptable behavior  
- Taking appropriate action in response to misconduct  
- Removing or rejecting contributions that violate this code  
- Banning contributors who repeatedly or severely violate the rules  

---

## Scope
This Code of Conduct applies to:

- GitHub issues and discussions  
- Pull requests  
- Project chat or communication channels  
- Any public or private interaction related to the project  

---

## Consequences
Violations may result in:

- Warnings  
- Temporary or permanent bans  
- Rejection of contributions  
- Removal from project spaces
